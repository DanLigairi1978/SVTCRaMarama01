# **App Name**: Ra Marama

## Core Features:

- Home Page with Featured Projects: Display featured projects from Firestore with summaries and a call to action to 'Discover Our Projects.' Includes a live feed of the latest news updates.
- About Us Section: Showcase the SVTC's history, purpose, vision, mission, values, and a dedicated section for Ra Marama Great Hall.
- Our Work Showcase: Present ongoing core programs, strategic plans, and donor-funded projects with partner information, all pulled dynamically from Firestore.
- Get Involved Guide: Explain membership criteria, detail the need for donor funds, and feature a partner logo gallery with thank you messages.
- Interactive Contact Form: A contact form with fields for name, email, subject, and message. The form sends data to a Firebase Function to securely forward to the SVTC administration.
- Dynamic Content Management: Admin panel to allow administrators to create and edit posts and other dynamic content.
- Geographic Tool: Provide geographic location info by reasoning about text using geographic data about the SVTC location and output a location section in the page.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), inspired by the ocean and sky of Cakaudrove, evokes trust and stability.
- Background color: Very light desaturated blue (#F0F4FF) for a clean, professional feel that doesn't distract from the content.
- Accent color: Earthy Green (#4CAF50), analogous to blue, symbolizing growth, nature, and harmony.
- Headline Font: 'Poppins', sans-serif. Body Font: 'PT Sans', sans-serif.
- Use culturally relevant icons that represent SVTC's values.
- Mobile-responsive design with a clear, intuitive navigation for all devices.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
